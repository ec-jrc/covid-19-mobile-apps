The charts in this page are basic descriptives about COVID-19 related mobile applications (apps), based on manual analysis results.

The chart legends list the data categories that are appearing in each chart.
Data categories can be switched on and off by clicking on the respective label(s) in the legend.
The actual numbers for each data category on a graph are visible when hovering the data category on that graph.
